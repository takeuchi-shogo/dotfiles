# GitHub Review Workflow Usage

## Typical Prompts

```text
$github-review-workflow を使って、この branch の PR comments を整理して
```

```text
$github-review-workflow を使って、落ちている GitHub Actions を見て fix plan まで出して
```

```text
$github-review-workflow を使って、この PR の CI と unresolved comments を順に片付けて
```

## Ask-Back Template

```text
先に3点だけ確認します。

1. PR comments と failing CI のどちらを見ますか。
2. current branch の PR でよいですか。
3. まず状況整理だけにしますか、それとも修正 plan まで進めますか。
```
