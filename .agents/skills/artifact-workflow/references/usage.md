# Artifact Workflow Usage

## Typical Prompts

```text
$artifact-workflow を使って、この内容から提案書を作って
```

```text
$artifact-workflow を使って、まず PDF として整えたい
```

```text
$artifact-workflow を使って、この内容をスライドにして
```

```text
$artifact-workflow を使って、今の desktop の screenshot を撮って
```

## Ask-Back Template

```text
先に4点だけ確認します。

1. 最終成果物は何ですか。docx / pdf / slides / screenshot のどれですか。
2. 元になる内容はありますか。text / existing file / reference deck / running app のどれですか。
3. 保存先やファイル名の希望はありますか。
4. 見た目の fidelity は重要ですか。
```
