# WezTerm キーバインド

Leader キー: `Ctrl+Q`（2秒タイムアウト）

## 基本操作

| キー | 動作 |
|------|------|
| `Ctrl+A` | nvim でプロンプトを編集（Claude Code 向け） |
| `Ctrl+X` | コピーモードに入る |
| `Ctrl+Shift+P` | コマンドパレット |
| `Ctrl+Shift+Space` | Quick Select |
| `Cmd+F` | 検索モード |
| `Ctrl+U` | 文字セレクタ（絵文字等） |
| `Ctrl+R` | 設定を再読み込み |
| `Ctrl+K` / `Cmd+K` | スクロールバックをクリア |
| `Cmd+L` | デバッグオーバーレイ |
| `Shift+Enter` | 改行を送信（Claude Code 向け） |
| `Alt+¥` | バックスラッシュを送信 |

## ターミナル Toggle

| キー | 動作 |
|------|------|
| `Cmd+Ctrl+J` | 下部ターミナルペインの表示/非表示 |
| `Ctrl+Shift+D` | 右側ターミナルペインの表示/非表示 |

## ウィンドウ

| キー | 動作 |
|------|------|
| `Alt+Enter` | フルスクリーン切替 |
| `Cmd+N` | 新規ウィンドウ |
| `Cmd+H` | アプリケーション非表示 |
| `Cmd+M` | ウィンドウ最小化 |
| `Cmd+Shift+]` | 常に最前面に表示 |

## ペイン（Leader プレフィックス）

| キー | 動作 |
|------|------|
| `Leader` → `d` | 上下に分割 |
| `Leader` → `r` | 左右に分割 |
| `Leader` → `x` | ペインを閉じる |
| `Leader` → `h/j/k/l` | ペイン移動（左/下/上/右） |
| `Leader` → `z` | 直前のコマンドと出力をコピー |
| `Leader` → `Z` | ペインズーム切替 |
| `Leader` → `a` | ターミナル履歴を nvim で表示 |
| `Leader` → `s` | 設定モード（ペインリサイズ） |

## ペイン（直接キー）

| キー | 動作 |
|------|------|
| `Ctrl+Shift+Z` | ペインズーム切替 |
| `Ctrl+Shift+S` | ペイン選択 |
| `Shift+Backspace` | 左ペインへ移動 |
| `Ctrl+Shift+J/K/L` | 下/上/右ペインへ移動 |

## タブ

| キー | 動作 |
|------|------|
| `Ctrl+Tab` | 次のタブ |
| `Ctrl+Shift+Tab` | 前のタブ |
| `Cmd+1` ~ `Cmd+9` | タブを番号で切替 |
| `Cmd+T` | 新規タブ |
| `Cmd+W` | タブを閉じる |

## ワークスペース

| キー | 動作 |
|------|------|
| `Leader` → `w` | ワークスペース一覧 |
| `Leader` → `W` | 新規ワークスペース作成 |
| `Leader` → `$` | ワークスペース名変更 |
| `Leader` → `p` | カスタムランチャーメニュー |
| `Ctrl+Cmd+N` | 次のワークスペース |
| `Ctrl+Cmd+P` | 前のワークスペース |
| `Alt+;` | タブ一覧（fzf） |

## フォント

| キー | 動作 |
|------|------|
| `Cmd+0` | フォントサイズリセット |
| `Cmd++` | フォントサイズ拡大 |
| `Cmd+-` | フォントサイズ縮小 |

## スクロール

| キー | 動作 |
|------|------|
| `Shift+PageUp/Down` | スクロール |
| `Alt+Ctrl+P` | 半ページ上 |
| `Alt+Ctrl+N` | 半ページ下 |
| `Alt+[` | 前のプロンプトへ |
| `Alt+]` | 次のプロンプトへ |

## クリップボード

| キー | 動作 |
|------|------|
| `Cmd+C` | コピー |
| `Cmd+V` | ペースト |

## その他

| キー | 動作 |
|------|------|
| `Cmd+,` | nvim で WezTerm 設定を編集 |
| `Cmd+.` | man ページビューア（fzf + nvim） |

## コピーモード（vim ライク）

| キー | 動作 |
|------|------|
| `h/j/k/l` | カーソル移動 |
| `0` / `^` / `$` | 行頭 / 行頭（非空白） / 行末 |
| `w` / `b` / `e` | 単語移動 |
| `f/F/t/T` | 文字ジャンプ |
| `;` / `,` | ジャンプ繰返し / 逆方向 |
| `v` / `V` / `Ctrl+V` | 文字 / 行 / 矩形選択 |
| `y` | ヤンク（コピー） |
| `Y` | ヤンク + 翻訳 |
| `g` / `G` | スクロールバック先頭 / 末尾 |
| `H` / `M` / `L` | ビューポート上 / 中央 / 下 |
| `Ctrl+U` / `Ctrl+D` | 半ページ上 / 下 |
| `Ctrl+E` / `Ctrl+Y` | 1行スクロール上 / 下 |
| `/` | 検索モード |
| `Ctrl+N` / `Ctrl+P` | 次 / 前の検索結果 |
| `[` / `]` | 前 / 次の入力ゾーン |
| `z` | セマンティックゾーン選択 |
| `Escape` / `q` | コピーモード終了 |

## 設定モード（Leader → s）

| キー | 動作 |
|------|------|
| `h/j/k/l` | ペインサイズ調整 |
| `Escape` / `q` | モード終了 |

## マウス

| 操作 | 動作 |
|------|------|
| トリプルクリック | セマンティックゾーンを選択 |
