# AeroSpace キーバインド

タイル型ウィンドウマネージャ。

## Main モード

### フォーカス / ウィンドウ移動

| キー | 動作 |
|------|------|
| `Alt+H/J/K/L` | フォーカス移動（左/下/上/右） |
| `Alt+Shift+H/J/K/L` | ウィンドウ移動（左/下/上/右） |

### リサイズ

| キー | 動作 |
|------|------|
| `Alt+Shift+-` | リサイズ縮小（-50） |
| `Alt+Shift+=` | リサイズ拡大（+50） |

### レイアウト

| キー | 動作 |
|------|------|
| `Alt+/` | tiles レイアウト切替（horizontal/vertical） |
| `Alt+,` | accordion レイアウト切替（horizontal/vertical） |

### ワークスペース切替

| キー | 動作 |
|------|------|
| `Alt+1` ~ `Alt+9` | ワークスペース 1-9 |
| `Alt+A` ~ `Alt+Z` | ワークスペース A-Z |
| `Alt+Tab` | 前のワークスペースに戻る |

### ウィンドウをワークスペースへ移動

| キー | 動作 |
|------|------|
| `Alt+Shift+1` ~ `Alt+Shift+9` | ワークスペース 1-9 へ移動 |
| `Alt+Shift+A` ~ `Alt+Shift+Z` | ワークスペース A-Z へ移動 |

### モニター

| キー | 動作 |
|------|------|
| `Alt+Shift+Tab` | ワークスペースを次のモニターへ移動 |

### モード切替

| キー | 動作 |
|------|------|
| `Alt+Shift+;` | Service モードに入る |

## Service モード

| キー | 動作 |
|------|------|
| `Esc` | 設定リロード → Main に戻る |
| `R` | レイアウトリセット → Main に戻る |
| `F` | フローティング/タイリング切替 → Main に戻る |
| `Alt+Shift+H/J/K/L` | ウィンドウ結合（左/下/上/右） → Main に戻る |
