# /epd

EPD 統合ワークフローを開始する: $ARGUMENTS

epd スキルを使用して、アイデアから本番コードまでの一連のフローを実行する。

1. 引数あり: 引数をアイデアとして EPD フロー開始
2. 引数なし: 対話でアイデアを聞き出してから開始
