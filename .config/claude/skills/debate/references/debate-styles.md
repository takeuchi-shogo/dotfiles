# Debate Styles

## Quick (2-3 min)

- 1問のみ、各モデル1回回答
- 短い質問（1-2文）に適する
- 合意/相違のみ報告

## Thorough (5-10 min)

- 質問を2-3サブトピックに分解
- 各モデルがサブトピックごとに回答
- クロスリファレンス + 合意率算出

## Adversarial (10-15 min)

- 一方のモデルの回答を他方に批判させる
- 反論→再反論の1ラウンド
- 最終的に統合見解を生成

## Style Selection

| Situation | Recommended |
|-----------|------------|
| Yes/No の判断 | Quick |
| 技術選定 | Thorough |
| 設計の問題点発見 | Adversarial |
| 事実確認 | Quick (debate 不要、WebSearch 推奨) |
