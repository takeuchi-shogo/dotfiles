# Web Design Guidelines — Category Checklist

## 1. Animation
- [ ] Purpose-driven motion only
- [ ] Duration < 300ms for micro-interactions
- [ ] Respect prefers-reduced-motion

## 2. Typography
- [ ] System font stack or performance-optimized web fonts
- [ ] Readable line height (1.4-1.6)
- [ ] Max line length 60-80 characters

## 3. Color
- [ ] WCAG AA contrast ratios
- [ ] Dark mode support
- [ ] Consistent palette (max 5 colors + neutrals)

## 4. Layout
- [ ] Responsive breakpoints tested
- [ ] Consistent spacing system (4px/8px grid)
- [ ] Content-first hierarchy

## 5. Accessibility
- [ ] Keyboard navigable
- [ ] Screen reader tested
- [ ] Focus indicators visible

## 6. Performance
- [ ] Images optimized (WebP/AVIF)
- [ ] Lazy loading for below-fold
- [ ] Critical CSS inlined

## 7. Icons
- [ ] SVG preferred over icon fonts
- [ ] Consistent size and stroke weight
- [ ] Accessible labels
