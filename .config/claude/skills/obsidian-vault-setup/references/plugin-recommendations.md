# Obsidian Plugin Recommendations

## Essential (全ユーザー推奨)

| Plugin | Purpose |
|--------|---------|
| Templater | テンプレートエンジン |
| Dataview | メタデータクエリ (2026 maintenance mode、新規は **Bases** 推奨) |
| Bases | 公式コア DB ビュー (Obsidian 1.5+、Dataview 後継) |
| Calendar | デイリーノートナビゲーション |

> **Note**: Dataview は 2026 でメンテモード移行。公式 **Bases** プラグインが新標準。  
> 既存 Dataview 資産は維持しつつ、新規 dashboard は Bases ベースで設計を検討。  
> 詳細: `references/dashboard-template.md`
