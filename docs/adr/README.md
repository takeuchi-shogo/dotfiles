# Architecture Decision Records

本ディレクトリには ADR（Architecture Decision Records）を格納する。
Nygard 形式に従い、各 ADR は不変。状態変更は新しい ADR で Supersede する。

## ステータス

- **Accepted**: 現在有効な決定
- **Superseded by ADR-XXXX**: 新しい ADR に置き換え済み
- **Deprecated**: 非推奨（代替なし）
